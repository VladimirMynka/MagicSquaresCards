### Text
В общем виде произведение двух полумагических квадратов выглядит следующим образом:
```
sm(E₁, x₁, y₁, w₁, v₁) * sm(E₂, x₂, y₂, w₂, v₂) =
sm(
   3E₁E₂,
   3x₁w₂ + 3x₂w₁ - 3y₁v₂ + 3y₂v₁,
  -3x₁v₂ + 3x₂v₁ + 3y₁w₂ + 3y₂w₁,
    x₁x₂ -  y₁y₂ + 3w₁w₂ + 3v₁v₂,
   -x₁y₂ +  x₂y₁ + 3w₁v₂ + 3w₂v₁,
)
```
